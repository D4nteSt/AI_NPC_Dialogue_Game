# AI NPC Dialogue Game

Игровой прототип на Unity с системой диалогов NPC на основе нейросети.  
Проект демонстрирует взаимодействие игрока с персонажами через текстовый чат, где NPC отвечают с учетом своего характера, роли, текущего квеста, инвентаря игрока и истории диалога.

Данный проект является дипломной работой по теме:  
**«Разработка игрового приложения на Unity с системой диалогов NPC на основе нейросети»**

## Описание

Проект представляет собой демонстрационную игровую сцену, в которой игрок может вести свободный текстовый диалог с NPC.  
Ответы персонажей генерируются нейросетевой моделью через локальный backend-сервер.

AI-диалоговая система связана с игровыми механиками:

- квестовой системой;
- инвентарем;
- уведомлениями;
- контекстными реакциями NPC;
- правилами игровых последствий диалога.

При этом нейросеть отвечает только за естественную реплику персонажа, а игровые действия выполняются через контролируемую rule-based систему.

## Демонстрационная сцена

Текущая демонстрационная сцена выполнена в детективном сеттинге.

Сюжетный контекст:

> В городском архиве пропала важная печать. Игрок помогает инспектору Баттлеру провести первичный осмотр, получить разрешение на доступ к месту происшествия, найти улику и передать её для дальнейшего расследования.

Основные персонажи:

- **Инспектор Баттлер** — ведет дело и выдает поручение игроку;
- **Мира** — свидетельница, которая реагирует на ход расследования и найденные улики.

## Функционал

- Свободный текстовый диалог с NPC
- Генерация ответов NPC через AI backend
- Учет характера, роли и знаний NPC
- Учет статуса квеста и предметов в инвентаре
- История последних реплик в контексте диалога
- Многострочный ввод в чат
- Отправка сообщения по Enter
- Перенос строки через Shift + Enter
- Системные сообщения в отдельном стиле
- Определение намерения игрока
- Rule-based система игровых последствий диалога
- Старт квеста через диалог
- Выдача предметов через диалог
- Сдача предметов NPC
- Обновление статуса квеста
- Инвентарь с UI-отображением предметов
- Журнал квестов с UI-отображением статусов
- Уведомления о получении предметов и изменении квестов
